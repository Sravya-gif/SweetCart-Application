# SweetCart Automation Framework

This is a Selenium automation testing project developed using Java and TestNG.
It automates key functionalities of the SweetCart web application.

## Features

* Login and Logout testing
* Product validation
* Add to basket
* Checkout process
* Navigation testing

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven

## Framework Design

* Page Object Model (POM)
* Reusable methods and clean structure

## How to Run

* Open project in Eclipse/IDE
* Run using testng.xml
